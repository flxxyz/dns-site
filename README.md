# Private-DNS

一个运行在 **树莓派3b** 上，干净安全的DNS服务(基于pihole)

> 规则中可能存在***误拦截***，可以加白名单

## 上游DNS

### IPv4

- 8.8.8.8
- 1.1.1.1
- 223.5.5.5

### IPv6
- 2001:4860:4860:0:0:0:0:8888
- 2606:4700:4700::1111
